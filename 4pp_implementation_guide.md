## 1. 시스템 개요

### 1.1 목표

Signatone SP4-40085TFJ 4-point probe 헤드를 사용하여 ITO 및 박막의 **Sheet Resistance (Rs, Ω/□)**를 측정하는 PoC 시스템 구현.

### 1.2 스코프

| 포함 | 제외 |
|------|------|
| 전원 (12V + USB) | 스테퍼 모터/드라이버 |
| 정전류원 (LM334) | 리미트 스위치 |
| 전압측정 (ADS1220) | 모터 케이블링 |
| 입력 보호 (클램프) | |
| 전류 반전 (DPDT 릴레이) | |
| **(BOM에 있는) Linear Motion** | **Tier 2: 자동화 (이번 PoC 제외)** |

> [!CAUTION]
> **프로브 사용 시 필수 주의사항** (Gemini 피드백)
> - 측정 전 반드시 **샘플의 수평을 확인**하고 프로브를 **수직으로 내릴 것**
> - Signatone 프로브 팁은 매우 날카롭고 민감하므로 비스듬히 내리면 팁 손상 위험
> - **프로브 압력 Spec** (ChatGPT 피드백): 모델명 `085`는 **팁당 85g**을 의미합니다.
>   - 4팁 전체 하중: 약 **340g**
>   - Z축 기구는 이 하중을 지지하고 오버슈트 없이 부드럽게 하강해야 합니다.
>   - 수평 이동(드래그) 절대 금지! 팁 수명 직결.

## 2. 핵심 설계 원칙

> [!IMPORTANT]
> 아래 5가지 원칙을 반드시 준수해야 측정이 제대로 됩니다.

### 2.1 ADC 입력 전압을 레일 내에 유지

```
ADS1220 입력 안전 범위 (PGA bypass 시):
AVSS − 0.1V  ~  AVDD + 0.1V
   ↓              ↓
 -0.1V          3.4V  (3.3V 공급 기준)
```

**해결책**: 정전류원을 3.3V 레일 기반으로 구성 → 샘플 전위가 0~3.3V 내에 머무름

### 2.2 전류는 가정값이 아닌 실측값 사용

> [!CAUTION]
> LM334는 **온도 의존성 (+0.33%/°C)**이 있어 전류가 변합니다.
> 반드시 100Ω shunt로 전류를 실측하여 계산에 반영하세요.

```
I_actual = V_shunt / 100Ω
R_sample = V_sense / I_actual  ← 이렇게 해야 정확
```

### 2.3 전류 반전으로 오프셋 제거 (Delta Mode)

열기전력(Thermal EMF)과 ADC 오프셋을 제거하는 **가장 효과적인 방법**:

```
1. 전류 정방향 인가 → (settle 대기) → V_fwd 측정
2. 릴레이로 전류 방향 반전 → (settle 대기) → V_rev 측정
3. V_corrected = (V_fwd - V_rev) / 2
4. Rs = 4.532 × (V_corrected / I)
```

> [!TIP]
> **Settle Time** (ChatGPT 피드백)
> - 릴레이 스위칭 후 **100~300ms 대기** 권장
> - 스위칭 직후 **1~2개 샘플 버리고** 평균 계산하면 결과가 더 안정됨

### 2.4 구역 분리 (아날로그 vs 디지털/릴레이)

| 구역 | 전원 | 구성 요소 |
|------|------|----------|
| **아날로그 구역** | 3.3V only | ADS1220, 입력 보호, 100Ω shunt |
| **시끄러운 구역** | 12V | 릴레이 코일, 향후 모터 드라이버 |

| **모터 구동 시** | - | **측정 중 드라이버 EN=OFF (필수)** |
 
 → **물리적으로 분리** (스트립보드 2장 또는 거리 확보)

### 2.5 LM334 헤드룸 한계 (Dropout 주의)

> [!WARNING]
> **고저항 샘플 측정 시 필수 체크!** (Gemini 피드백)
> 
> 3.3V 전원 시스템 특성상 샘플 저항이 **수 kΩ 이상**으로 높아지면 
> LM334가 정전류를 유지하지 못할 수 있습니다 (Dropout).

```
LM334 동작 조건:
  V_headroom = V_supply - V_sample - V_shunt - V_LM334(min)
             = 3.3V - (I × R_sample) - (I × 100Ω) - 0.9V

  ↓ R_sample이 너무 크면 ↓
  V_headroom < 0 → LM334 Dropout → I_actual 급락!
```

**소프트웨어 모니터링 필수**:
- 매 측정마다 `I_actual = V_shunt / 100Ω` 계산
- 설정값 대비 **10% 이상 감소** 시 경고 출력
- 예: 1mA 설정인데 0.85mA 측정 → "Headroom Warning!"

**실제 한계 추정** (1mA 기준):
```
R_sample_max ≈ (3.3V - 0.1V_shunt - 0.9V_LM) / 1mA ≈ 2.3kΩ
```

> [!CAUTION]

### 4.4 DPDT 전류 반전 회로 (핵심!)

```
              ┌─────────────────────────────────────────────┐
              │              DPDT 릴레이 (TQ2-12V)           │
              │                                             │
              │   Pole A (소스 쪽)      Pole B (리턴 쪽)      │
              │   ┌───────────┐        ┌───────────┐        │
              │   │           │        │           │        │
   SHUNT_LO ──┼──►│   COM_A   │        │   COM_B   │◄───AGND│
              │   │     │     │        │     │     │        │
              │   │  ┌──┴──┐  │        │  ┌──┴──┐  │        │
              │   │ NC_A  NO_A│        │ NC_B  NO_B│        │
              │   │  │     │  │        │  │     │  │        │
              │   └──┼─────┼──┘        └──┼─────┼──┘        │
              └─────┼─────┼──────────────┼─────┼────────────┘
                    │     │              │     │
                    │     └──────────────┼─────┼──► I- Post
                    │                    │     │
                    └────────────────────┼─────┘
                                         │
                                         ▼
                                       I+ Post
```

**릴레이 배선 (정석)**:

| Pole | Pin | 연결 |
|------|-----|------|
| **A** | COM_A | SHUNT_LO (LM334 출력→션트 지난 뒤) |
| **A** | NC_A | I+ Post (Red) |
| **A** | NO_A | I- Post (Black) |
| **B** | COM_B | AGND (전류 리턴) |
| **B** | NC_B | I- Post (Black) |
| **B** | NO_B | I+ Post (Red) |

**동작 원리**:
- **릴레이 OFF (NC 접점)**: SHUNT_LO→I+, AGND→I- → **정방향 전류**
- **릴레이 ON (NO 접점)**: SHUNT_LO→I-, AGND→I+ → **역방향 전류**

**LM334 핀 연결**:
- **V+ (pin 1)**: 3V3_A 입력
- **R (pin 2)**: Rset 한쪽
- **V- (pin 3)**: Rset 다른쪽 + 전류 출력

## 5. 스트립보드 배치도

> [!NOTE]
> 스트립보드는 **세로 방향(열)으로 구리가 연결**되어 있습니다.
> 좌표: **가로 = 행 (A, B, C...)**, **세로 = 열 (1, 2, 3...)** — 체스판과 동일
> **트랙 컷 없이** 설계되었습니다.

> [!WARNING]
> **누설 전류 관리** (Gemini 피드백)
> - IN0/IN1 스트립(열 5, 6)은 **주변 스트립과 거리를 넓히거나** 사이 구멍을 **물리적으로 긁어내어** 절연 저항 극대화
> - **nA 단위의 누설 전류**도 mV 측정에 오차 유발 가능
> - **플럭스 청소 필수!** IPA(이소프로필 알코올)로 납땜 후 잔여 플럭스 완전 제거

## 6. 조립 순서 (Bring-up)

> [!TIP]
> **핵심 원칙**: 작은 단위로 테스트하면서 진행 → 문제 발생 시 원인 추적 용이

### Step 1: 라벨링 및 준비

- [ ] 패널 포스트에 **I+, V+, V-, I-** 라벨 부착
- [ ] 스트립보드에 테스트 포인트 위치 표시
- [ ] SOT23→DIP 어댑터에 BAT54A, BAT54C 납땜 (핀 방향 확인!)

**BAT54A/C 핀 확인 방법**:
```
SOT-23 패키지 (위에서 봤을 때, 핀1 마크 기준)
      ┌───┐
   1 ─┤   ├─ 3
      │   │
   2 ─┤   ├
      └───┘

BAT54A: pin1=Cathode1, pin2=Cathode2, pin3=Common Anode
BAT54C: pin1=Anode1, pin2=Anode2, pin3=Common Cathode

⚠️ 핀아웃 확인 필수: 구매한 정확한 MPN의 데이터시트를 기준으로 핀배열 확인 후 납땜!
```

### Step 2: 전원 필터 섹션 조립

조립 (섹션 5.2.1 배치도 기준):
- [ ] 10Ω 저항: 열2(3V3_DIG) ↔ 열3(3V3_A) (행 C)
- [ ] 0.1µF: 열1(AGND) ↔ 열3(3V3_A) (행 E)
- [ ] 1µF: 열1(AGND) ↔ 열3(3V3_A) (행 F)
- [ ] Arduino 3V3 → 열2(3V3_DIG), Arduino GND → 열1(AGND)

**검증**:
```
1. Arduino USB 연결
2. 멀티미터로 측정:
   - B1 (3V3_DIG): 3.3V ± 0.1V ✓
   - C3 (3V3_A): 3.3V ± 0.1V ✓ (거의 같아야 함)
```

### Step 3: ADS1220 SPI 통신 확인

연결:
- [ ] ADS1220 VDD/AVDD → 열3 (3V3_A)
- [ ] ADS1220 GND/AVSS → 열1 (AGND)
- [ ] SPI: SCK, MOSI, MISO, CS, DRDY → Arduino 해당 핀

**검증 코드**:
```cpp
// 간단한 SPI 통신 테스트
#include <SPI.h>

#define CS_PIN 10
#define DRDY_PIN 9

void setup() {
  Serial.begin(115200);
  SPI.begin();
  pinMode(CS_PIN, OUTPUT);
  pinMode(DRDY_PIN, INPUT);
  
  digitalWrite(CS_PIN, HIGH);
  delay(100);
  
  // ADS1220 리셋
  digitalWrite(CS_PIN, LOW);
  SPI.transfer(0x06); // RESET command
  digitalWrite(CS_PIN, HIGH);
  delay(100);
  
  // 레지스터 읽기 테스트
  digitalWrite(CS_PIN, LOW);
  SPI.transfer(0x20); // RREG, read Config Register 0
  byte reg0 = SPI.transfer(0x00);
  digitalWrite(CS_PIN, HIGH);
  
  Serial.print("ADS1220 Config Reg 0: 0x");
  Serial.println(reg0, HEX);
  // 기본값 0x00이 나와야 정상
}

void loop() {}
```

**예상 결과**: `Config Reg 0: 0x00`

### Step 4: 입력 보호 회로 조립

조립:
- [ ] 1kΩ: 열8(V+_RAW) → 열5(IN0) (행 F)
- [ ] 1kΩ: 열9(V-_RAW) → 열6(IN1) (행 G)
- [ ] BAT54A: pin3→열1(AGND), pin1/2→열5/6 (IN0/IN1)
- [ ] BAT54C: pin3→열3(3V3_A), pin1/2→열5/6 (IN0/IN1)
- [ ] ADS1220 AIN0 → 열5(IN0), AIN1 → 열6(IN1)

**검증**:
```
1. V+ 포스트와 V- 포스트를 쇼트
2. ADS1220 읽기 → 거의 0V (수 µV 이내)

3. 3.3V를 V+ 포스트에 1kΩ 통해 인가
   → IN0_NODE가 3.3V 근처, 클램프 동작으로 ADS1220 손상 없음

4. GND를 V+ 포스트에 1kΩ 통해 인가
   → IN0_NODE가 0V 근처, 클램프 동작
```

### Step 5: 정전류원 조립 (더미 부하 테스트)

조립:
- [ ] LM334: V+ → 열3(3V3_A), R → 열12, V- → 열13
- [ ] Rset: 열12(LM_R) ↔ 열13(LM_OUT)
  - **권장**: 바로 납땜 대신 **핀헤더 소켓(1x2)** 사용 (나중에 100µA 저항으로 교체 용이하게)
- [ ] 100Ω Shunt: 열13(LM_OUT) ↔ 열15(SHUNT_LO)
  > [!WARNING]
  > **저항 혼동 주의**: 10Ω (갈-검-검-금) 아니고 **100Ω (갈-검-검-검-갈 또는 101)**입니다!
  > 10Ω은 전원 필터용, 100Ω은 전류 측정용입니다. 바뀌면 전류 계산이 10배 틀어집니다.
- [ ] 더미 부하 저항 (1kΩ ~ 10kΩ)을 전류 경로에 연결

**검증**:
```
예상 전류: I = 67.7mV / 681Ω ≈ 99.4µA

Shunt 전압 측정: V_shunt = I × 100Ω ≈ 9.94mV
→ 멀티미터로 9~11mV 범위면 정상

더미 부하 (1kΩ) 양단 전압: V = I × R ≈ 0.1V
```

> [!NOTE]
> **헤드룸 확인**:
> - 100µA 일 때: 3.3V − 1V(LM334) − 10mV(Shunt) ≈ **2.29V** 여유
> - 1mA 일 때: 3.3V − 1V(LM334) − 100mV(Shunt) ≈ **2.20V** 여유 (약간 줄어듦)

### Step 6: 릴레이 구동 추가 (12V 섹션)

조립:
- [ ] 릴레이 코일 → 12V / PN2222A Collector
- [ ] 1N4148 플라이백 (코일 병렬, 역방향)
- [ ] PN2222A Base → 1kΩ → Arduino GPIO
- [ ] (추천) PN2222A Base → 10kΩ → GND (풀다운)
- [ ] PN2222A Emitter → 12V GND
- [ ] 470µF: 12V(+) ↔ 12V(-)
- [ ] 0.1µF: 12V(+) ↔ 12V(-) (필수)
- [ ] (옵션) 0.1µF: 릴레이 코일 핀 바로 양단 (스너버)

**검증**:
```cpp
#define RELAY_PIN 2

void setup() {

}

void loop() {
  digitalWrite(RELAY_PIN, HIGH);
  delay(1000);
  digitalWrite(RELAY_PIN, LOW);

```

- 릴레이 on/off 시 ADC 읽기값이 튀는지 확인
- 튀면: 디커플링 보강, Star Ground 재검토

## 7. 소프트웨어 구현

### 7.1 ADS1220 설정 (권장값)

```cpp
// ADS1220 레지스터 설정
// Config Register 0
//   MUX = AIN0/AIN1 (차동)
//   GAIN = 1 (PGA bypass) → VCM(공통모드) 제약 회피, 회로 단순화 (ChatGPT 권장)
//   PGA_BYPASS = 1 (입력 범위 최대화)
byte config0 = 0b00000001; // MUX=00, GAIN=000, PGA_BYPASS=1

// Config Register 1
//   ⚠️ DR/CM 비트 해석 주의! (ChatGPT 피드백)
//   비트 배치: [DR2 DR1 DR0 MODE1 MODE0 CM TS x]
//   - DR[2:0] = 000 → 20 SPS (Normal mode)
//   - MODE[1:0] = 00 → Normal mode  
//   - CM = 1 → Continuous conversion
//   - TS = 0 → Disabled
byte config1 = 0b00000100; // DR=000(20SPS), MODE=00, CM=1(Continuous), TS=0
// byte config1 = 0b00000000; // 싱글샷 원하면: CM=0

// Config Register 2
//   ⚠️ VREF 설정 주의! (ChatGPT 피드백)
//   ADS1220 데이터시트 VREF[1:0] 정의:
//     00 = 내부 2.048V ← PoC 기본 권장!
//     01 = 외부 REFP0/REFN0 (배선 필요)
//     10 = 외부 REFP1/REFN1
//     11 = AVDD–AVSS (ratiometric)
//   50/60 = Simultaneous rejection
//   PSW = Open
//   IDAC = Off
byte config2 = 0b00010000; // VREF=00(내부2.048V), 50/60=01, PSW=0, IDAC=000

// Config Register 3
//   기본값 사용
byte config3 = 0b00000000;

// 설정 적용 후:
// SPI.transfer(0x08); // START/SYNC command (연속 변환 시작)
```

### 7.2 전류 반전 측정 루틴 (Delta Mode)

```cpp
#include <SPI.h>

#define CS_PIN 10
#define DRDY_PIN 9
#define RELAY_PIN 2
#define SHUNT_R 100.0  // Ohms

// ADS1220 관련 함수들...

float readVoltage(byte mux) {
  // MUX 설정 업데이트 (Config0)
  byte cfg0 = (config0 & 0x1F) | mux; // MUX bits[7:4] update
  digitalWrite(CS_PIN, LOW);
  SPI.transfer(0x40); // WREG Config0
  SPI.transfer(cfg0);
  digitalWrite(CS_PIN, HIGH);
  
  // START/SYNC (연속 모드 재동기화)
  digitalWrite(CS_PIN, LOW);
  SPI.transfer(0x08);
  digitalWrite(CS_PIN, HIGH);
  
  // DRDY 대기 (Polling with Timeout)
  unsigned long start = millis();
  while (digitalRead(DRDY_PIN) == HIGH) {
    if (millis() - start > 100) { // 100ms 타임아웃
       Serial.println("Error: DRDY Timeout!");
       return 0.0;
    }
  }
  
  // 데이터 읽기 (RDATA)
  digitalWrite(CS_PIN, LOW);
  SPI.transfer(0x10); // RDATA
  long raw = 0;
  raw |= (long)SPI.transfer(0x00) << 16;
  raw |= (long)SPI.transfer(0x00) << 8;
  raw |= (long)SPI.transfer(0x00);
  digitalWrite(CS_PIN, HIGH);
  
  // 첫 샘플 Discard (안정성 확보, 옵션)
  // 필요한 경우 위 읽기 루틴을 한 번 더 반복
  
  // 부호 확장 (24-bit to 32-bit signed)
  if (raw & 0x800000) raw |= 0xFF000000;
  
  // 전압 변환 (Internal VREF 2.048V, Gain=1)
  // V = (raw / 2^23) * 2.048
  return (float)raw * 2.048 / 8388608.0;
}

struct Measurement {
  float V_corrected;
  float I_actual;
  float R_sample;
  float Rs;
};

Measurement measureWithReversal() {
  Measurement m;
  
  // 1. 정방향 전류
  digitalWrite(RELAY_PIN, LOW);
  delay(500);  // 안정화 대기
  
  float V_fwd = readVoltage(0x00);  // V sense
  float V_shunt_fwd = readVoltage(0x30);  // Shunt
  
  // 2. 역방향 전류
  digitalWrite(RELAY_PIN, HIGH);
  delay(500);  // 안정화 대기
  
  float V_rev = readVoltage(0x00);  // V sense
  float V_shunt_rev = readVoltage(0x30);  // Shunt
  
  // 3. 계산
  m.V_corrected = (V_fwd - V_rev) / 2.0;
  
  // 전류도 반전 평균 (더 정확)
  float I_fwd = V_shunt_fwd / SHUNT_R;
  float I_rev = abs(V_shunt_rev) / SHUNT_R;  // 부호 반대
  m.I_actual = (I_fwd + I_rev) / 2.0;
  
  m.R_sample = m.V_corrected / m.I_actual;
  
  // Correction Factor (샘플 크기/형상 보정)
  // ITO 1인치 중앙 측정 전용 상수 (NBS 표준 기반, 원형 근사)
  // F = 4.47044 (CF ≈ 0.986)
  float F_GEOM = 4.47044;  
  
  m.Rs = F_GEOM * m.R_sample;  // Sheet Resistance
  
  // 4. 릴레이 원위치
  digitalWrite(RELAY_PIN, LOW);
  
  return m;
}
```

### 7.3 Ohmic Contact 검증 (전류 스윕)

```cpp
bool verifyOhmicContact() {
  // 여러 전류값에서 측정하여 R이 일정한지 확인
  // Rset 교체 또는 DAC 제어 필요 (향후 확장)
  
  float R_values[4];
  float I_test[] = {0.0002, 0.0005, 0.0008, 0.001};  // 200µA ~ 1mA
  
  for (int i = 0; i < 4; i++) {
    // 해당 전류로 설정하고 측정
    // (실제로는 Rset 교체 또는 외부 DAC 필요)
    Measurement m = measureWithReversal();
    R_values[i] = m.R_sample;
  }
  
  // R값 변화 확인
  float R_avg = (R_values[0] + R_values[1] + R_values[2] + R_values[3]) / 4.0;
  float max_deviation = 0;
  
  for (int i = 0; i < 4; i++) {
    float dev = abs(R_values[i] - R_avg) / R_avg * 100.0;
    if (dev > max_deviation) max_deviation = dev;
  }
  
  // 5% 이상 변동이면 접촉 불량 가능성
  if (max_deviation > 5.0) {
    Serial.println("WARNING: Non-ohmic contact suspected!");
    Serial.print("Max deviation: ");
    Serial.print(max_deviation);
    Serial.println("%");
    return false;
  }
  
  return true;
}
```

## 9. 측정 공식 및 보정

### 9.1 기본 공식

```
기본 저항:
  R_sample = V_sense / I_actual

Sheet Resistance (무한 평판 근사):
  Rs = (π / ln2) × R_sample
     = 4.532 × R_sample  [Ω/□]
```

### 9.2 전류 반전 보정 (Delta Mode)

```
V_corrected = (V_fwd - V_rev) / 2

이점:
  - ADC 오프셋 제거
  - 열기전력 (Thermal EMF) 제거
  - 접촉 비대칭 보상
```

### 9.3 샘플 크기 보정 계수 (Correction Factor)

> [!WARNING]
> **중요**: 4.532 공식은 **샘플이 프로브 간격의 40배 이상** 클 때만 유효합니다.

```
프로브 간격: s = 40 mil = 1.016 mm
최소 샘플 크기 (4.532 유효): 40 × 1.016 mm ≈ 40 mm

작은 샘플에서는 보정 계수(CF) 적용 필요:
  Rs = CF × (π / ln2) × (V / I)
```

**NBS/NIST TN 199 표준 값 (원형 샘플, 중앙 측정)**:

- 1인치 샘플 / 40mil 프로브 (d/s ≈ 25)
- **F = 4.47044** (CF ≈ 0.986)
- 가이드의 표는 **이 값 하나로 고정**합니다. (다른 크기 샘플은 별도 계산 필요)

> [!NOTE]
> ITO가 "정사각형" 1인치라도, 1차 근사로 위 원형 F값을 사용합니다. 정확도를 높이려면 표준 시료로 캘리브레이션 하세요.

**직사각형 샘플**:
- SEMI MF84 표준 또는 Smits 논문 참조
- 가장자리에서 최소 4s (4 × 프로브 간격) 거리 확보 권장

### 9.4 두께 효과

4-point probe는 기본적으로 **박막 (두께 ≪ 프로브 간격)** 가정:

```
박막 조건: t < s/5 (약 0.2 mm 이하)

후막의 경우:
  - 벌크 저항률 ρ = Rs × t 로 환산
  - 또는 두께 보정 적용 필요
```
